# Variables

Define variables with one letter

```ruby
sentence="This is a good sentence."
s="This is a good sentence." #Better
```

Multiple variable assignement

```ruby
a=2017
b=50
c="Fish"

a,b,c=2017,50,"Fish" #Better
```

Puts variables

```ruby
a="Anana"
puts "My name is #{a}"

$a="Anana"
puts "My name is #$a" #Better
```
